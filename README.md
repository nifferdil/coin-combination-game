# Scrabble

##### Epicodus Week One Assignment, 08/6/2015

##### Jennifer Morkunas & Archana Bengale

## Description
A web app where users can type in a dollar amount and find out how much change they have in quarters, dimes and nickels.

## Setup

Clone this repository:
```
$ cd ~/Desktop
$ git clone
https://github.com/nifferdil/coin-combination-game.git
$ cd coin-combination-game
```

Navigate back to the directory where this repository has been cloned and run gradle:
```
$ gradle run
```
